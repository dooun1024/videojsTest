初始化代码

videojs(videoElement, {
  crossOrigin: 'use-credentials',   // 原生 <video> 元素
  sources: [{ src: '...', type: 'video/mp4' }],
  html5: {
    vhs: { withCredentials: true },  // VHS (HLS/DASH)
    hls: { withCredentials: true }   // 兼容旧版
  }
});

如何验证 Cookie 是否携带

  1. 打开浏览器 DevTools（F12）
  2. 切换到 Network 标签
  3. 点击播放，找到 sample.mp4 请求
  4. 查看 Request Headers 是否包含 Cookie 字段
  5. 查看请求是否包含 withCredentials: true 标记
注意：Cookie 跨域生效需服务端设置 Access-Control-Allow-Origin（不能为 *）和 Access-Control-Allow-Credentials: true。
